Бир хил билимлар графини уч хил усулда кўриш мумкин. Three ways to view the same knowledge graph:

## 1. 🖥️ Тўлиқ интерактив граф / Full interactive graph
**[Graph.html](https://bismailov.github.io/qora-apelsin-kb/static/graph.html)** — тўлиқ экранли, тугунлар боғланишлар сонига қараб катталашади, мавзу бўйича рангланган, қидирув ва қўшни тугунларни ёритиш. Full-screen, node size = connections, coloured by theme, search + neighbour highlight. *(Graphify built-in.)*

## 2. 🔬 Граф тадқиқотчиси / Graph explorer (filters, layout)
**[Gephi Lite explorer](https://lite.gephi.org/?file=https%3A%2F%2Fbismailov.github.io%2Fqora-apelsin-kb%2Fstatic%2Fgraph.gexf)** — фильтрлар, ўлчам/ранг атрибут бўйича, тугун панели, тартиб (layout). Filters, size/colour by attribute, node panel, re-layout. Opens our GEXF in the free web Gephi Lite.

**[Retina explorer](https://ouestware.gitlab.io/retina/beta/#/graph/?url=https%3A%2F%2Fbismailov.github.io%2Fqora-apelsin-kb%2Fstatic%2Fgraph.gexf)** — соддалаштирилган улашиладиган граф кўрувчи. A streamlined shareable graph viewer (loads the same GEXF).

*(GraphML варианти / GraphML variant: [Gephi Lite + GraphML](https://lite.gephi.org/?file=https%3A%2F%2Fbismailov.github.io%2Fqora-apelsin-kb%2Fstatic%2Fgraph.graphml).)*

## 3. 🕸️ Викитўр граф / Wiki + mini-graph
Ушбу сайтнинг ўнг панелидаги граф — ҳар бир саҳифа учун локал боғланишлар. The mini-graph in this site's right sidebar shows local links per page. [[index|← Бош саҳифа / Home]]
